# Sprint 2: Modern Data Transformation with dbt

## Goal
Use dbt Core to transform raw data into a clean Star Schema

## Learning Objectives
- Understand dbt Core and transformation concepts
- Create dbt models and understand the ref() function
- Design and implement Star Schema (facts vs dimensions)
- Build data quality tests

## Daily Tasks
**Day 1:** dbt Core installation and project setup
**Day 2:** Create staging models from raw data
**Day 3:** Design star schema for e-commerce data
**Day 4:** Build dimension and fact models
**Day 5:** Add dbt tests and documentation

## Success Criteria
- [ ] dbt project initialized
- [ ] Star schema designed and implemented
- [ ] Data quality tests added
- [ ] Documentation generated
